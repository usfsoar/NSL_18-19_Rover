# NSL_18-19_Rover
Code for the rover for NASA Student Launch 2018-19.

## Instructions
Download all the files and compile nsl_rover_18-19.ino using the Arduino IDE.

*Pins for Adafruit DRV8871 Motor 1 and Motor 2*

```
MOTOR   INPUT    PIN
MOTOR1  INPUT1   8
MOTOR1  INPUT2   9
MOTOR2  INPUT1   10
MOTOR2  INPUT2   11
```
